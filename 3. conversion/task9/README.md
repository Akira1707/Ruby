# Task 09 – Markdown → HTML (Groovy)

## Описание
Программа конвертирует Markdown в HTML, поддерживает:
- Заголовки: #, ##, ###
- Жирный текст: **text**
- Курсив: *text*
- Ссылки: [text](url)
- Встроенный код: `code`

## Входные данные
Файл `example.md`

## Выходные данные
Файл `output.html`

## Запуск
groovy task9.groovy example.md
