# Task 01 (Rust) - Параллельная загрузка изображений

## Описание
Программа читает список URL из файла `input_urls.txt` и имитирует их загрузку в несколько потоков.
Для каждого URL создается запись в JSON-логе с информацией:
- Поток, который обработал URL
- Время обработки в миллисекундах
- Статус ("success")

## Входные данные
Файл `urls.txt`

## Выходные данные
Файл `log.json`:
[
  {
    "status": "success",
    "thread": 0,
    "time_ms": 502,
    "url": "http://example.com/image1.jpg"
  }
]

## Как запускать

- Скомпилируйте: cargo build 

- Запустите: cargo run 

- Проверьте файл log.json
