# Task 02 (Elixir) – Многопоточный подсчёт файлов

## Описание
Программа рекурсивно обходит каталог и подсчитывает количество файлов и строк по расширениям.
Каждая часть обрабатывается параллельно в отдельном процессе Elixir.

## Входные данные
Текстовый файл, содержащий путь к каталогу, например:
./test_folder

## Выходные данные
Файл `file_count.json` с подсчётом файлов и строк по расширениям, например:
{
  ".txt": {"files": 2, "lines": 5}
}

## Запуск
Запустите команду:
mix run lib/task2.ex input_dir.txt
