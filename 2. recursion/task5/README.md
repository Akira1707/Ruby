# Task 05 – Построение дерева каталогов (OCaml)

## Описание
Программа рекурсивно обходит каталог и строит JSON-структуру каталога, включая файлы и подкаталоги.

## Входные данные
Аргумент командной строки: путь к каталогу, например:
./test_folder

pgsql
Копировать код

## Выходные данные
Файл `dir_tree.json`, пример:
{
    "type": "dir",
    "name": "test_folder",
    "children": ...
}

## Запуск
Скомпилируйте:
ocamlc -o task5.exe task5.ml

Запустите:
./task5.exe ./test_folder
Результат будет в dir_tree.json
