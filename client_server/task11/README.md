# Task 11 – TCP-клиент (Python)

## Постановка задачи
Создать TCP-клиент на Python, который подключается к серверу, написанному на Ruby (Task 10),  
и получает от него ответ на команду `QUOTE`.

## Входные данные
- IP-адрес и порт сервера (вводятся как аргументы командной строки)
- Сервер должен быть заранее запущен и иметь файл `input.txt`

## Выходные данные
- Ответ от сервера (цитата), сохраняется в указанный выходной файл `output.txt`.

## Пример запуска
ruby client_server/task10/task10_server.rb input.txt 4000
python client_server/task11/task11_client.py 127.0.0.1 4000 output.txt
