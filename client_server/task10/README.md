# Task 10 - TCP сервер (Echo / Quote)

## Постановка задачи

Программа создаёт TCP сервер, который принимает команды от клиентов:

- `ECHO:message` — возвращает сообщение обратно.  
- `QUOTE` - возвращает случайную цитату из файла.  

Каждый клиент обслуживается в отдельном потоке.  
Все команды можно логировать в файл `ouput.txt`.
