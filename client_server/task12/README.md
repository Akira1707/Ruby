# Task 12 - Мини-REST-сервер (WEBrick)

## Постановка задачи

Программа реализует **простой REST-сервер на Ruby** с использованием библиотеки **WEBrick**.  
Сервер принимает HTTP-запросы `POST` на путь `/convert` и возвращает JSON-ответ с количеством символов и слов в переданном тексте.

---

## Входные данные

- **HTTP POST-запрос** с телом в формате JSON:

{ "text": "Пример текста для анализа" }

## Выходные данные
Ответ сервера в формате JSON: { "length": , "words":  }
