# Task 09 - Markdown -> HTML (упрощённый парсер)

## Постановка задачи

Программа преобразует Markdown-файл в HTML.  
Поддерживается базовый синтаксис:

- Заголовки (# … ######)  
- Жирный текст (**bold**)  
- Курсив (*italic*)  
- Встроенный код (`code`)  
- Ссылки ([text](url))  

Абзацы оборачиваются в <p>.
