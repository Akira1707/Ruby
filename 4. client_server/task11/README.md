# Task 11 (Lua) - TCP-клиент 

## Описание
Программа на Lua, которая подключается к TCP-серверу, отправляет команды и получает ответы.  
Используется стандартная библиотека `socket`.

Поддерживаются команды:
- `ECHO:<текст>` -> возвращается тот же текст
- `QUOTE` -> сервер возвращает случайную цитату
- Неизвестные команды ->`"Unknown command"`

## Входные данные
`client_input.txt`

## Выходные данные
`client_output.txt`

## Запуск
lua task11.lua
