# Task 10 (D)- TCP-сервер Echo/Quote 

## Описание
Программа реализует TCP-сервер на D, который принимает команды от клиентов:

- `ECHO:<текст>` → возвращает тот же текст
- `QUOTE` → возвращает случайную цитату из файла `quotes.txt`
- Любая другая команда → `"Unknown command"`

Сервер многопоточный: каждый клиент обслуживается в отдельном потоке с помощью `core.thread.Thread`.

## Файлы
- `quotes.txt` – список цитат, по одной на строку

## Запуск
1. Скомпилируйте:
dmd task10.d

2. Запустите:
./task10.exe

4. Сервер будет слушать на `127.0.0.1:3000`. Подключитесь через telnet, nc или TCP-клиент.

## Пример работы
Подключение через `telnet`:
telnet 127.0.0.1 3000
ECHO:Hello Server
QUOTE
