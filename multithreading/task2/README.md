# Task 02 – Многопоточный подсчёт файлов

## Постановка задачи

Программа выполняет **многопоточный подсчёт строк в файлах**, указанных во входном списке.  
Каждый поток обрабатывает часть файлов из очереди, считает количество строк в каждом файле и фиксирует время обработки.

Результаты сохраняются в **JSON-файл** с подробной информацией о каждом обработанном файле.
