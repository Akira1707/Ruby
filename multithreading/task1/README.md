# Task 01 – Параллельная загрузка изображений (имитация)

## Постановка задачи

Программа имитирует **параллельную загрузку изображений** по списку URL-адресов, распределяя задачи между несколькими потоками.  
Для каждой "загрузки" фиксируется время выполнения, номер потока и отметка времени завершения.  
Результаты сохраняются в JSON-файл.
