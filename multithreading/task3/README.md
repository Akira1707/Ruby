# Task 03 – Анализ данных датчиков (Sensor Data)

## Постановка задачи

Программа выполняет **параллельную обработку файлов с данными датчиков**.  
Каждый CSV-файл содержит измерения температуры и влажности.  
Несколько потоков независимо читают файлы, вычисляют статистику и формируют общий отчёт в формате JSON.
