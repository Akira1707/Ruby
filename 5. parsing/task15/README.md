# Задача 15 (Python) — Подсчёт хэштегов 

## Описание
Программа подсчитывает **частоту использования хэштегов** в текстовых постах и сохраняет результат в формате **JSON**.  

Каждый хэштег начинается с символа `#` и может содержать буквы, цифры и символ подчеркивания `_`.

## Вход
Текстовый файл `posts.txt`, где каждая строка представляет собой отдельный пост.  

## Выход
JSON-файл `hashtags.json` с частотой использования каждого хэштега.  

## Запуск
python task15.py
