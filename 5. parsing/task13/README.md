# Task 13 (ML) - Анализ логов веб-сервера

## Описание задачи

Программа на языке **Standard ML (SML)** анализирует лог-файл веб-сервера и находит **топ-5 IP-адресов**, которые чаще всего встречаются в записях.
Формат входных данных — текстовый лог, где каждая строка начинается с IP-адреса клиента.  

## Как работает программа

1. **Читает** лог-файл построчно.  
2. **Извлекает первый токен** каждой строки (до первого пробела).  
3. **Проверяет**, является ли токен корректным IPv4-адресом.  
4. **Подсчитывает**, сколько раз каждый IP встречается.  
5. **Сортирует** адреса по количеству вхождений (по убыванию).  
6. **Выводит** первые 5 IP-адресов в виде JSON в выходной файл.

## Формат вывода (JSON)
Пример результата:
[
    {"ip": "192.168.0.1", "count": 12},
    {"ip": "10.0.0.2", "count": 9},
    {"ip": "127.0.0.1", "count": 4},
    {"ip": "172.16.0.3", "count": 3},
    {"ip": "192.168.1.5", "count": 2}
]
## Запуск
sml task13.sml
LogAnalyzer.analyze "access.log" "top_ips.json";
