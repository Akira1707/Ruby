# Task 04 – Факториал и числа Фибоначчи (рекурсия + мемоизация)

## Постановка задачи

Программа вычисляет **факториалы** и **числа Фибоначчи** для заданных чисел с использованием **рекурсии**.  
Для чисел Фибоначчи применяется **мемоизация**, чтобы избежать лишних повторных вычислений и переполнения стека.

---

## Входные данные

- **JSON-файл** с числами для вычислений.  
Пример `input.json`:

```json
{
  "factorials": [5, 7],
  "fibs": [10, 20]
}
