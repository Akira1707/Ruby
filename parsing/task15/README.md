# Task 15 – Подсчёт хэштегов

## Описание
Программа подсчитывает хэштеги в постах соцсетей, выводит топ N популярных.

## Входные данные
Текстовый файл `input.txt` с постами в формате `user<TAB>content`.

## Выходные данные
JSON-файл `output.json` с ключами:
- `total_posts` — количество постов
- `top_hashtags` — массив топ N хэштегов и их количества

## Запуск
```bash
ruby task15_hashtag_counter.rb input.txt output.json 10
