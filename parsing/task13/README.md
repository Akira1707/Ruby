# Task 13 – Анализ логов веб-сервера

## Описание

Парсинг веб-логов в формате:

```
[timestamp] IP METHOD PATH STATUS
```

Цель: извлечь время, IP, метод, путь, статус и подсчитать топ-5 IP по числу запросов.

## Входные данные

Текстовый лог-файл, например `input.log`:

```
[2025-10-09 14:23:11] 192.168.1.2 GET /index.html 200
[2025-10-09 14:23:12] 192.168.1.3 POST /login 302
```

## Выходные данные

JSON-файл `output.json` с:

* `entries` — массив всех записей
* `top_ips` — массив топ-5 IP с количеством запросов

## Запуск

```bash
ruby task13_log_parser.rb input.log output.json
```
