# Task 14 – Извлечение контактов (email и телефон)

## Описание
Парсинг текста для извлечения email и телефонных номеров. Удаляются дубликаты. Результат сохраняется в JSON.

## Входные данные
Текстовый файл, например `input.txt`.

## Выходные данные
JSON-файл `output.json` с массивами `emails` и `phones`.

## Запуск
```bash
ruby task14_email_extractor.rb input.txt output.json
